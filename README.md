# And the Tortoise Won

Short audiovisual interactive fiction game based on the Tortoise and the Hare fable.

Made with:
* [Godot](https://github.com/godotengine/godot) 4.5.1
* [GIMP](https://gitlab.gnome.org/GNOME/gimp-data)
* [Furnace](https://github.com/tildearrow/furnace)

Running and exporting requires Godot.
